│  HELGRIND (Race Condition Detection):                            │
│    Terminal 1:                                                   │
│      valgrind --tool=helgrind \                                  │
│        --log-file=helgrind_output.txt ./server                   │
│    Terminal 2:                                                   │
│      for i in {1..100}; do curl localhost:8080/test.html; done   │
│    Stop server, check: helgrind_output.txt                       │
│                                                                  │
│  THREAD SANITIZER (Data Race Detection):                         │
│    make clean                                                    │
│    CFLAGS='-fsanitize=thread -g -O1' \                           │
│    LDFLAGS='-fsanitize=thread' make                              │
│    ./server 2> tsan_output.txt                                   │
│    (run some requests, stop, check tsan_output.txt)              │
│                                                                  │
│  VALGRIND (Memory Leaks):                                        │
│    Terminal 1:                                                   │
│      valgrind --leak-check=full \                                │
│        --log-file=valgrind_memcheck.txt ./server                 │
│    Terminal 2:                                                   │
│      ab -n 1000 -c 50 http://localhost:8080/test.html            │
│    Stop server, check: valgrind_memcheck.txt                     │
│                                                                  │
